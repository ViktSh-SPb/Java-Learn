Чтобы использовать **лямбда-выражение** для сортировки списка строк в **обратном порядке** с помощью `Comparator`, можно сделать так:
## 1. Пример с `List<String>` и `sort`
```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        List<String> names = new ArrayList<>(Arrays.asList("Anna", "Bob", "Charlie"));

        // Сортировка в обратном порядке с помощью лямбды
        names.sort((s1, s2) -> s2.compareTo(s1));

        System.out.println(names);
    }
}
```
**Вывод:**
```
[Charlie, Bob, Anna]
```
## 2. Пояснение
- `s1` и `s2` — два элемента списка, которые сравниваются
- `s2.compareTo(s1)` вместо `s1.compareTo(s2)` → **обратный порядок**
- Метод `List.sort` принимает `Comparator` (функциональный интерфейс)
- Лямбда реализует метод `compare(T o1, T o2)`
## 3. Использование `Comparator.reverseOrder()`
Альтернатива без ручной лямбды:
```java
names.sort(Comparator.reverseOrder());
```
Эквивалент:
```java
names.sort((s1, s2) -> s2.compareTo(s1));
```
## 4. С использованием Stream API
Если нужен новый список:
```java
List<String> sortedNames = names.stream()
    .sorted((s1, s2) -> s2.compareTo(s1))
    .toList();
```
### Коротко
- Лямбда для `Comparator`: `(s1, s2) -> s2.compareTo(s1)`
- Можно использовать встроенный `Comparator.reverseOrder()`
- Подходит для `List.sort` и `Stream.sorted`


### 5. Выбрать имена и зарплаты сотрудников, у которых зарплата выше средней по компании


```sql
SELECT name, salary FROM employees WHERE salary > (SELECT AVG(salary) FROM employees);
```

### 6. Обновить должность всех сотрудников в отделе "Sales" на "Sales Manager"



### 9. Выбрать имена и зарплаты сотрудников, у которых зарплата выше средней по их отделу


```sql
SELECT name, salary 
FROM employees e1
WHERE salary > (
    SELECT AVG(salary) 
    FROM employees e2 
    WHERE e1.department = e2.department
);
```

### 10. Обновить должность всех сотрудников в отделе "Sales" на "Sales Manager", если их текущая должность не "Sales Manager"


```sql
UPDATE employees 
SET position = 'Sales Manager' 
WHERE department = 'Sales' AND position <> 'Sales Manager';
```

### 12. Удалить всех сотрудников, которые не вносили вклад в проект за последние 6 месяцев

(предполагается наличие поля `last_contribution_date`)


```sql
DELETE FROM employees WHERE last_contribution_date < DATE_SUB(CURDATE(), INTERVAL 6 MONTH);
```

```sql
from patients
```

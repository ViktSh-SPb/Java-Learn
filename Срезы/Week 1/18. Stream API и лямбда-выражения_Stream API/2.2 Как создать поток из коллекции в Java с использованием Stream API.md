В Java для создания потока (`Stream`) из коллекции используется метод **`stream()`** (или `parallelStream()` для параллельного потока).
## 1. Из `List` или `Set`
```java
import java.util.*;
import java.util.stream.Stream;

public class Main {
    public static void main(String[] args) {
        List<String> names = List.of("Anna", "Bob", "Charlie");

        // Создание последовательного потока
        Stream<String> stream = names.stream();

        // Пример: вывод элементов
        stream.forEach(System.out::println);

        // Для параллельного потока
        Stream<String> parallelStream = names.parallelStream();
        parallelStream.forEach(System.out::println);
    }
}
```
## 2. Из массива
```java
String[] arr = {"A", "B", "C"};
Stream<String> stream = Arrays.stream(arr);
stream.forEach(System.out::println);
```
## 3. Из `Map`
Поток создаётся из:
- ключей (`keySet()`)
- значений (`values()`)
- записей (`entrySet()`)
```java
Map<Integer, String> map = Map.of(1, "A", 2, "B");

map.keySet().stream().forEach(System.out::println);
map.values().stream().forEach(System.out::println);
map.entrySet().stream().forEach(System.out::println);
```
## 4. Создание потока из `Stream.of()`
Можно создать поток из отдельных элементов:
```java
Stream<String> stream = Stream.of("A", "B", "C");
stream.forEach(System.out::println);
```
## 5. Важные моменты
1. Поток **не хранит данные**, а оперирует источником.
2. Поток **однократного использования** — после терминальной операции нельзя использовать снова.
3. Для **параллельной обработки** используем `parallelStream()`.
## 6. Короткая шпаргалка

|Источник коллекции|Метод для потока|
|---|---|
|List, Set|`stream()` / `parallelStream()`|
|Массив|`Arrays.stream(array)`|
|Map|`map.keySet().stream()`, `map.values().stream()`, `map.entrySet().stream()`|
|Произвольные элементы|`Stream.of(elem1, elem2, ...)`|
